Application Software Engineer

Hyderabad, Telangana, India Graduate Program

Job description

Help build software requirements for software projects Design and implement applications as needed for others within SIG and other end user customers Perform complete testing of each release, and handle end user testing, prior to deployment Working directly with software engineering peers to understand and fix issues Identify and address root cause problems Quickly adept and integrate new tools, systems, and functionality Participate in continuous improvement of software design and implementation Develop and maintain software and documentation Promptly respond to inquiries or issues Assist in system installation and configuration Provide end user training when vital Support software solutions, responding to urgent off-hours support calls when significant impacts are encountered

Experience

  • Knowledge in developing and supporting applications for the Windows and Linux Operating Systems
  • Knowledge in developing and supporting applications for the Windows and Linux Operating Systems
  • Knowledge in coding and supporting applications, primarily Java. Additional experience with Python, C and Perl is a plus.
  • Knowledge in crafting relational databases (I.e. MongoDB, PostgreSQL, MySQL)
  • To be able to write sophisticated SQL commands and stored procedures

Qualifications

  • Excellent verbal and written communication skills (English)
  • Excellent organization skills Optimally multi-task between projects
  • Effectively collaborate with other Micron team members
  • Strong problem solving and analytical skills Passion for excellence and innovation Independent learner
  • Enforce to team guidelines and implement standard methodologies in design and implementation
  • Proactive Transparent and Diligent Flexible and willing to put in extra effort.
